# 🐄 Centralized Data Mart & Gestalt BI Dashboard — Thesis Project (BIB Lembang)

![Dashboard Preview](screenshots/dashboard_preview.png) 

## 📌 Project Overview
Project Tugas Akhir ini bertujuan untuk mentransformasi sistem pelaporan di Balai Inseminasi Buatan (BIB) Lembang dengan membangun **Centralized Data Mart** dan merancang Business Intelligence Dashboard menggunakan pendekatan **Design Thinking** dan **Prinsip Visualisasi Gestalt**. Solusi ini mengeliminasi masalah *data silos* serta inkonsistensi data antar divisi menjadi *insight* yang terpadu, interaktif, dan mudah dipahami oleh pengambil keputusan.

## 🛠️ Tech Stack & Tools
* **Data Integration & Modeling:** Power Query, Data Modeling
* **Database / Data Mart:** Relational Database / Centralized Data Mart Architecture
* **BI & Visualization:** Power BI
* **Methodology:** Kimball Lifecycle (Data Mart), Design Thinking, Gestalt Visual Perception Principles, User Acceptance Testing (UAT).

## 🚀 Key Features & Methodologies
1. **Centralized Data Mart Architecture:** Mengintegrasikan aliran data dari **4 departemen operasional** yang sebelumnya terfragmentasi menjadi satu *Single Source of Truth* (SSOT).
2. **Gestalt-Driven Dashboard Design:** Mengimplementasikan prinsip Gestalt (seperti *Proximity*, *Similarity*, dan *Continuity*) dipadukan dengan *Design Thinking* untuk membangun **4 dashboard interaktif**. Desain ini secara drastis mengurangi *cognitive load* pengguna.
3. **Rigorous Quality Validation:** Melibatkan *User Acceptance Testing* (UAT) secara menyeluruh untuk memastikan akurasi arsitektur data dan kesesuaian visual dengan kebutuhan *stakeholder*.

## 📈 Business Impact & Quantitative Results
* Menyatukan data operasional dari **4 departemen berbeda** ke dalam satu arsitektur terpusat.
* Berhasil **menurunkan inkonsistensi data lintas departemen hingga 90%**, menciptakan standarisasi pelaporan yang jauh lebih akurat.
* Mencapai tingkat kepuasan dan adopsi yang tinggi dengan **lulus 100% pada tahap User Acceptance Testing (UAT)** oleh pihak manajemen BIB Lembang.
